## Razões para remoção:

1. **Baixa importância preditiva**: Features com importância menor que 0.1% da importância total.
2. **Alta variabilidade entre modelos**: Features cujo coeficiente de variação entre diferentes modelos é maior que 1.5.
3. **Redundância**: Features altamente correlacionadas (r > 0.8) com outras de maior importância.
